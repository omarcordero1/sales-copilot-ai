# Guía de contribución

¡Gracias por tu interés en contribuir a este proyecto! Las contribuciones son bienvenidas y valoradas. Ya sea reportando un error, sugiriendo una mejora o enviando código, tu ayuda hace que la comunidad sea mejor.

## 📌 Cómo reportar problemas

Si encuentras un error o tienes una sugerencia, por favor abre un **issue** en el repositorio. Incluye:

- Una descripción clara del problema o mejora.
- Pasos para reproducir el error (si aplica).
- Versión del sistema operativo, Python, y dependencias relevantes.
- Capturas de pantalla o fragmentos de código si son útiles.

## 🧠 Cómo sugerir nuevas funcionalidades

Las ideas son bienvenidas. Antes de crear un issue, revisa si ya existe una propuesta similar. Si no es así, abre un issue con la etiqueta `enhancement` y describe:

- El problema que soluciona o la oportunidad que aprovecha.
- Una posible implementación o alternativas consideradas.
